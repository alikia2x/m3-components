## Introduction

M3 Components offers a comprehensive suite of UI tools for developing beautiful, usable products
that adhere to [Material Design 3](https://m3.material.io) and its update, [M3 Expressive](https://m3.material.io/blog/building-with-m3-expressive).
Both are part of Material Design, Google's robust and adaptable design system.
